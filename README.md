## ⚡ Installation

```bash
# Clone repository
git clone https://github.com/Nancy-1311/Nancy-FullStack-Developer---Portfolio.git

# Navigate to project
cd Nancy-FullStack-Developer---Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```
